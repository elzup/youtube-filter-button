# Privacy Policy — YouTube Filter Button

Last updated: 2026-03-17

## Overview

YouTube Filter Button is a browser extension that adds time-period filter buttons to YouTube search results. This policy explains how the extension handles user data.

## Data Collection

This extension does **not** collect, store, or transmit any personal data.

## Permissions

### `storage`

Used solely to save the user's filter button preferences (e.g., which filter buttons to display). This data is stored locally via `chrome.storage.sync` and is never sent to any external server.

## Third-Party Services

This extension does not communicate with any external servers or third-party services. It operates entirely within the user's browser on youtube.com.

## Changes to This Policy

If this policy is updated, the changes will be posted on this page with an updated date.
